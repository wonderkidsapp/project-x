# HPWS Backend

This is the backend service for the HPWS application, written in Python as per project requirements.

## Purpose
- Data processing and Analytics
- API Services for standardizing logic if needed
- interacting with Supabase for data analysis

## Setup
1. Create virtual environment: `python -m venv venv`
2. Activate venv: `source venv/bin/activate` (or `venv\Scripts\activate` on Windows)
3. Install dependencies: `pip install -r requirements.txt`

## Running
`python src/main.py`
